Relação entre a qualidade do sabor e da acidez do café

Este código tem como objetivo criar um gráfico de dispersão interativo. O gráfico mostra a relação entre o sabor e a acidez de diferentes amostras de café, permitindo que o usuário analise os dados para o país de sua preferência. As cores atribuídas a cada empresa permitem identificar as amostras de café de diferentes produtores. Os tooltips fornecem informações específicas de cada ponto quando o mouse passa sobre eles. O título e os rótulos dos eixos fornecem uma descrição gráfica, e as cores foram escolhidas de forma a não distrair a atenção das distribuições dos pontos no gráfico de dispersão.

Presença dos países no dataset

Este código gera um gráfico de barras interativo que exibe a contagem dos países de origem do dataset. Cada barra no gráfico representa um país, e sua altura é proporcional ao número de ocorrências desse país na amostra. As cores das barras são escolhidas a partir de uma paleta predefinida com o propósito de garantir uma distinção visual entre os países presentes no gráfico. O gráfico também inclui elementos interativos, como tooltips, que fornecem informações adicionais quando o mouse é posicionado sobre uma barra. Além disso, o código utiliza um RangeSlider, que permite ao usuário ajustar o intervalo da contagem dos países exibidos no gráfico, e um botão de reset que reverte essas modificações e uma ferramenta que permite ao usuário selecionar as barras em questão e desfocar nas não selecionadas. Também foi adicionado um contorno para destacar as barras escolhidas. O título e os rótulos dos eixos fornecem uma descrição gráfica, e as cores foram escolhidas de forma a produzir uma visualização esteticamente agradável para os grids do gráfico.

Variação da qualidade geral do café por País

O código gera um gráfico de box plot interativo que mostra a distribuição dos pontos da nota geral da qualidade do café por país. Cada box plot no gráfico representa um país e seu tamanho indica a variação dos pontos de qualidade dentro desse país. As cores atribuídas a cada país permitem identificar visualmente as amostras de qualidade geral do café. Foram adicionados elementos interativos para melhorar a visualização. A ferramenta ResetTool permite ao usuário restaurar as configurações originais do gráfico. Foi acrescentado um menu dropdown que permite ao usuário filtrar os países exibidos no gráfico, facilitando a análise exploratória dos dados de países específicos de interesse. Em relação aos elementos visuais, o código personaliza o estilo da grade do gráfico para torná-lo esteticamente agradável. A escolha das cores, tanto para as caixas quanto para as linhas, foi feita com o objetivo de garantir melhor legibilidade e contraste, garantindo que as informações sejam transmitidas de forma clara e coesa. No gráfico gerado, quando apenas um traço é visível no país, indica que há pouca informação fornecida pelo conjunto de dados para criar uma representação gráfica adequada. Os whiskers que estão por dentro das caixas podem indicar possíveis anomalias nos dados fornecidos.
